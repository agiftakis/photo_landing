01 // What It Does

See everything. Edit anything.

Most photo tools show you a handful of "safe" fields and hide the rest. Photo Meta Editor reads every single tag in a photo using a bundled copy of ExifTool (the industry-standard metadata engine — no separate install needed) and lets you edit any field that's actually writable.

  • Reads ALL metadata — EXIF, IPTC, XMP, GPS, ICC profiles, and manufacturer MakerNotes (Apple, Samsung, Canon, Sony, Nikon, Google & more).
  • Edit writable fields — captions, keywords, ratings, copyright, dates, camera tags, and more.
  • ADD GPS coordinates — put a location on photos that have none. Just type decimal latitude/longitude; the app handles the N/S/E/W details for you.
  • Automatic originals backup — the first time you edit a photo, a pristine copy is saved as filename_original. Nothing is ever lost.
  • Safe by design — read-only/computed fields are locked so you can't corrupt a file. Only the green fields are editable.
› The actual app — 117 tags read, green fields editable, GPS ready to write.
02 // How To Use It

Five steps, under a minute

01
Download the app Click the red Download for Windows button. You'll get a single PhotoMetaEditor.exe — no installer, no setup wizard.
02
Run it (accept the SmartScreen prompt) Double-click the .exe. On first launch Windows shows the blue "protected your PC" pop-up — click More info → Run anyway. Totally safe (see the red banner up top).
03
Open a photo Hit Open Photo and pick any image. Every tag it contains loads instantly into the table.
04
Edit fields or add GPS Type new values into any green (editable) field. To geotag a photo with no location, enter decimal latitude & longitude in the Add GPS box.
05
Save changes Click Save. Your edits are written to the photo, and a pristine _original backup is kept automatically the first time you touch a file.
